Part 3:
Building - Features
and Configuration

This part highlights many of Drupal's fundamental features that all developers need to understand. Each feature has its own configuration and use cases that can be set up per Drupal application. Readers will understand content structures, adding support for multiple languages, users, roles, permissions, and search. Configuring engagement through contact forms will be covered. And, dynamic displays with Drupal Views and Display Modes promote content reuse and build upon the structured content covered.

This part has the following chapters:

  • Chapter 8, Content Structures and Multilingual
  • Chapter 9, Users, Roles, and Permissions
  • Chapter 10, Drupal Views and Display Modes
  • Chapter 11, Files, Images, and Media
  • Chapter 12, Search
  • Chapter 13, Contact Forms
